Markdown

小怪 · 2019-11-26 · 次阅读

简介

Markdown是一种纯文本格式的标记语言。通过简单的标记语法，他可以使普通文本内容具有一定格式。

一、标题

设置标题只需在前面加上#，
一个#是一级标题（最大的），两个#是二级标题，以此类推 ，一直到六级标题。
在书写数记得在符号后加一个空格。

下面是示例：

    # 这是一级标题
    ## 这是二级标题
    ### 这是三级标题
    #### 这是四级标题
    ##### 这是五级标题
    ###### 这是六级标题

二、字体

  • 斜体

需要斜体的文字左右分别用一个*号括起来

  • 加粗

需要加粗的文字左右分别用两个*号括起来

  • 斜体加粗

需要斜体加粗的文字左右分别用三个*号括起来

  • 删除线

需要加删除线的文字左右分别用连个~~号括起来

下面是示例：

    *这是斜体*
    **这是粗体**
    ***这是斜体加粗***
    ~~这是删除线文字~~

三、引用

在引用文字前加>即可。引用也可以嵌套，这个我也不太清楚
貌似加的>>及>>>>没有任何变化

下面是示例：

    > 这是引用的文章

四、分割线

三个或三个以上的-+*都可以
…..我一般打四个…

下面是示例：

    ----
    ++++
    ****

五、图片

大概就是这么写…

下面是示例：

    ![alt](图片地址 "title")


    恩.  alt就是图片下面的文字相当于对图片的解释；
         title就是图片的标题，鼠标移动到图片上显示的内容

六、超链接

看下面示例

下面是示例：

    [超链接名](超链接地址 "title")

    title 跟上面差不多，可加可不加，

Markdown本身不支持连接在新页面打开，，若要在新页面打开的话需要用HTML语言的a标签代替。

七、列表

无序列表

无序列表用(-+*)中的任何一种都可以

下面是示例：

    + 这是无序列表
    - 这是无序列表
    * 这是无序列表

有序列表

需要在数字后加点

下面是示例：

    1. 有序列表
    2. 有序列表
    3. 有序列表

列表的嵌套

上一级和下一级之间敲两个空格就可以了
不同网页或浏览器好像不一样…
拿有序列表为例：

    1. 有序列表
        - 无序列表

八、表格

应该是这样使用

下面是示例：

    表头|表头|表头
    ---|：--：|---：
    内容|内容|内容
    内容|内容|内容

    第二行分割表头和内容
    - 没什么用是为了对其
    - 表示文字默认居左
    - 两边加： 表示文字居中
    - 右边加： 表示文字居右

九、代码

单行代码两边用一旁边的点号括起来
多行用三个点好括起来

下面是示例：

    `我是一行代码`



    ```
        我是代码
        我是多行
        我是点
    ```
